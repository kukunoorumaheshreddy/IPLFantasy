📋 How to get your league data

  1. Go to fantasy.iplt20.com and log in
  2. Open browser DevTools: press F12 → go to the Console tab
  3. Copy the contents of update-data.js (in the same folder as this file) and paste into the console
  4. Press Enter and wait — a JSON file will download automatically
  5. Move it to the master-snapshots/ folder, then refresh this page
