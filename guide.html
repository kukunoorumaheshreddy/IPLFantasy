🏟️ What Is This?

A live dashboard for our Big Blue Championship IPL Fantasy league. It auto-updates after each match — just visit:

kukunoorumaheshreddy.github.io/IPLFantasy/dashboard.html

Someone from the league needs to run a quick script after each match to pull the latest data. Here's how.

After Each Match (The Only Required Step)

After Pull Match Data

Do this once points are finalized — usually 30–60 min after the match ends.

Go to fantasy.iplt20.com and log in

Open fantasy.iplt20.com in Chrome/Edge. Make sure you're logged in to your IPL Fantasy account.

Open the browser console

Press F12 (or right-click → Inspect), then click the Console tab.

Paste the extraction script

Open update-data.js from the repo (view on GitHub), copy the entire file, paste it into the console, and press Enter.

If Chrome warns "Don't paste code you don't understand" — type allow pasting and press Enter first, then paste the script.

Wait ~30–60 seconds

The script fetches data for all 12 league members across all completed matches. You'll see progress in the console.

When done: ✅ Done! Downloading ipl-fantasy-v3-master-gd8.json

Send the downloaded JSON file

The file auto-downloads to your Downloads folder. It's named ipl-fantasy-v2-master-gd{N}.json where N is the latest match number.

Option A: If you have git access to the repo, move the file into master-snapshots/ and push:

git add master-snapshots/ && git commit -m "Match N data" && git push

Option B: Just send the file to Mahesh — he'll upload it.

You must be logged in to fantasy.iplt20.com in the same browser tab. The script uses your session to access the league API.
The script fetches everything from scratch each time — no need to keep old files or load previous data. Just paste and go.

🔧 Troubleshooting

"No completed gamedays" or empty data

You're not logged in. Go to fantasy.iplt20.com, log in, then run the script again in the same tab.

Chrome says "Don't paste code you don't understand"

Type allow pasting in the console and press Enter. Then paste the script.

Script seems stuck / no download after 2 minutes

Check the console for red errors. Most likely your session expired — refresh the page, log in again, then re-paste the script.

Dashboard shows "No data loaded"

The JSON file hasn't been pushed to the repo yet. Make sure the file is in the master-snapshots/ folder and pushed to GitHub.

Points look slightly different from the fantasy app

The script accounts for boosters (power-ups), but some edge cases may cause small differences. These are cosmetic and don't affect rankings.

🧠 How It Works (For the Curious)

update-data.js

  1. Fetches the IPL match schedule to find all completed matches
  2. For each league member: calls the overall-get API to get all match data in one shot
  3. Detects booster usage (FREE_HIT, DOUBLE_POWER, etc.) and calculates bonus points
  4. Builds cumulative rankings, transfer usage, and per-match breakdowns
  5. Downloads one self-contained JSON file with everything

Dashboard

Pure HTML + JavaScript — no server, no install. Hosted free on GitHub Pages. Auto-loads the latest JSON from the master-snapshots/ folder.

Features: Standings, Prizes, Per-Match Leaderboard, Rank Progression, Points Race (bar + line animation), Rank Heatmap.
